阿里云服务器部署web项目

发表于 2018-06-11
字数统计: 550 | 阅读时长 ≈ 2

Java web学习一段时间自己做了两个项目，发布到云服务器上实现访问。一实现Linux命令的熟悉，二尝试操作实际的项目发布过程，能配置一个域名。

前期准备

在阿里云服务器的网站上申请一台云服务器，本次抱着尝试的心态，没有计较服务器的性能。不过我也不懂，开始申请的服务器里面什么都没有，需要自己安装jdk和tomcat、mysql。在windows端远程连接云服务器，官网推荐的是putty，putty不能上传文件，故换做SecureCRT。

因为JDK，MYSQL，TOMCAT的安装过程中需要从网上下载部分支持包才可以继续，所以要提前安装下载好依赖 。

上传JDK、Mysql、Tomcat的安装包，创建文件夹（alt+p）

安装JDK

Jdk的安装路径在/usr/local/java，创建该目录mkdir -p /usr/local/java

解压 tar -zxvf jdk-7u71-linux-i586.tar.gz -C /usr/local/java

配置环境变量

vim命令

输入“vim 文件名”进入“一般模式”
按下“i”从一般模式，进入“插入模式”
按下“esc”从“插入模式”退出到“一般模式”
在“一般模式”下，输入“:wq”，退出编辑

安装MySQL

参考阿里云的文档

https://yq.aliyun.com/articles/285398

安装tomcat

上传tomcat的安装文件

创建tomcat的安装路径 mkdir -p /usr/local/tomcat

解压 tar -zxvf apache-tomcat-7.0.57.tar.gz -C /usr/local/tomcat

设置防火墙

启动tomcat

定位tomcat的安装路径 /usr/local/apache-tomcat-7.0.57
